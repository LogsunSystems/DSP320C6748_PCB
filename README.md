### DSP320C6748 DSP DVK Board
![DSP IC](.image/DSP320C6748.png)
The DVK board is based on the DSP chip from *Texas Instruments* DSP chip **DSP320C6748**. 

## Introduction
**DSP320C6748 DVK** is a scalable evaluation platform designed for high-performance applications requiring *real-time signal processing* and *embedded analytics*.

## Specifications
- **Processor:-** TSM320C6748 DSP with fixed and floating point core running at performance levels up to 456MHz
- **RAM:-**
- **Flash:-** 2Gbit, 16-bit Bus, Parallel Flash memory

- **Video Coded:-**

- **Interfaces:-**
